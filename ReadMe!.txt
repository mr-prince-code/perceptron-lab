# Perceptron Lab: Robot Librarian

## Overview
Implementation of a Perceptron classifier that acts like a robot librarian sorting books into fiction/non-fiction.

## Files
- `perceptron_lab.py`: Main Python code
- `bonus_challenge.py`: The bonus challenge code
- `learning_progress.png`: Exercise 2 plot
- `bonus_challenge.png`: Bonus challenge plot
- `iris_learning.png`: Exercise 5 plot

## Results
- The Perceptron learned to classify books with 100% accuracy.
- Decision boundary visualizations show how the model separates classes.
